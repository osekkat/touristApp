# What's New Template

Use this template for App Store and Play Store release notes.

---

## Template Format

```
Version X.Y.Z

• [Feature]: [Brief benefit]
• [Feature]: [Brief benefit]
• Performance improvements and bug fixes
```

---

## Example: Version 1.0.0 (Initial Release)

### English
```
Welcome to Marrakech Guide!

• Know Fair Prices: Check any quoted price instantly
• Navigate Offline: Compass always points home, no internet needed
• Speak Darija: 28 phrases with Arabic script and pronunciation
• Curated Places: 49 personally vetted spots, not overwhelming lists

Works completely offline. No subscriptions, no ads, no account required.
```

### French
```
Bienvenue sur Marrakech Guide !

• Connaissez les Vrais Prix : Vérifiez instantanément tout prix proposé
• Naviguez Hors Ligne : La boussole pointe toujours vers votre riad
• Parlez Darija : 28 phrases avec écriture arabe et prononciation
• Lieux Sélectionnés : 49 endroits vérifiés, pas de listes interminables

Fonctionne entièrement hors ligne. Sans abonnement, sans pub, sans compte.
```

## Guidelines

1. **Keep it short**: 3-4 bullet points maximum
2. **Lead with benefits**: "Check prices instantly" not "Added price checking feature"
3. **Be specific**: "12 new places" not "More content"
4. **End with technical**: Bug fixes go last
5. **Match tone**: Friendly but professional
6. **Localize fully**: Don't machine-translate, adapt for audience
